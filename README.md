# Volt-Orb Bowling Shop

A premium bowling e-commerce web app built with Vue 3, Vite, TypeScript, Pinia, and Vue Router.

## Tech Stack

- **Framework:** Vue 3 + Vite
- **Language:** TypeScript (strict mode)
- **State Management:** Pinia
- **Routing:** Vue Router
- **Styling:** SCSS with CSS variables
- **Backend:** Fake Store API

## Local Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

## AI Tool Usage

AI tools were used during development for scaffolding, boilerplate generation, and small refactors. All AI-generated code was manually reviewed, aligned with the project architecture, and TypeScript types were enforced throughout.
